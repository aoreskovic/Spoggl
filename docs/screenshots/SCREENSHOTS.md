# Screenshots to capture

Take these in Super Productivity with the Spoggl plugin open.
Recommended window width: ~1400px. Plugin panel ~300px wide.

## 01-main-view.png
Full plugin view with:
- Some entries in the entry list (left side)
- Day view timeline visible (right side) with 2-3 time blocks
- Timer NOT running (shows ▶ Start button)

## 02-timer-running.png
Close-up of the omnibar area:
- Timer running (shows elapsed time like "01:23:45" in red/active)
- Task name filled in the search field with a Jira chip (e.g. "PROJ-42")
- ■ Stop button visible

## 03-day-view-timeline.png
Day view timeline with:
- 3-4 completed time blocks at different hours
- Hour labels on the left (8:00, 9:00, 10:00...)
- "Now" line visible
- At least one synced (green) and one pending block

## 04-overlap-columns.png
Two entries that overlap in time displayed in separate side-by-side columns.
Tip: start a second timer while one entry overlaps the current time, then stop.
Both will appear side by side in the timeline.

## 05-task-sections.png
The task list area showing all three sections:
- ▼ Today (with 1-2 tasks)
- ▼ Assigned to me (with 2-3 Jira tasks)
- ▼ Other Tasks (with a few tasks)
One section should be collapsed (showing ▶ instead of ▼)

## 06-settings.png
Settings overlay open showing:
- Jira config fields (blur/redact the actual API token)
- Auto-detect from SP, Test connection, and Save buttons
- Pin layout radio options
- Font size dropdown

## 07-finish-day.png
After clicking "Finish Day" — either:
- The green success snack: "X entries synced!"
- Or the retry dialog if some entries failed (shows "Retry failed" button)

## 08-dark-mode.png
Same layout as 01-main-view.png but with SP set to dark mode.
Shows the plugin automatically following SP's dark theme.
